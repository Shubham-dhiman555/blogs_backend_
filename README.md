Blogs Backend API (Express + Prisma + PostgreSQL)

A RESTful backend API for a blogging platform built using Express.js, Prisma ORM, and PostgreSQL. It supports authentication and full CRUD operations for blogs.

🚀 Features
🔐 User Authentication (JWT)
📝 Blog CRUD (Create, Read, Update, Delete)
🧩 Prisma ORM (type-safe queries)
🗄️ PostgreSQL Database
⚡ REST API using Express.js
🛡️ Error handling & validation
🛠️ Tech Stack
Backend: Node.js, Express.js
Database: PostgreSQL
ORM: Prisma
Authentication: JWT
